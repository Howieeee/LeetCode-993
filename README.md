# LeetCode-993
## Cousins in Binary Tree

- 此題使用BFS (Breadth-First Search)

- 給定一個binary tree,且 node 的值都不重複
- 輸入兩個值(x,y), 最後回傳(x,y)是否為 Cousins
   (不同Parent, 相同Depth即為Cousins)
- root的depth = 0 , 每個depth=k的node 的child depth = k+1
